# xD4000Tool (v0.2)

SoMove-like starter for xD4000 VFD using Modbus TCP.

## Modules
- Parameters (tree + search + grouped lists)
- Operate (control panel + live monitor)
- Diagnostics (faults + history)
- Scope/Trend (basic)
- Project compare/backup-restore (basic)

Build using GitHub Actions workflow and download artifact.
